WPrez allows you to make beautiful presentations (in the form of "Prezi.com") right from your WordPress hosted website or blog.

== Description ==

WPrez uses impress.js and WordPress allowing you to create awesome presentations similar to "Prezi.com". They  run 100% in the browser so you don't need any software to view them.

== Installation ==

1. Unzip the plugin file and upload `wprez` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin options and start creating presentations.
